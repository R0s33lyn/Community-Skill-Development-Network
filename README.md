# Community Skill Development Network

A decentralized learning platform that connects mentors and learners through blockchain-verified skill development courses with peer validation and expert certification.

## Features

- **Course Creation**: Mentors can design and publish skill development programs
- **Peer Learning**: Learners can enroll and participate in community-driven education
- **Expert Certification**: Lead educators provide quality assurance and accreditation
- **Progress Tracking**: Complete learning journey records on the blockchain

## Smart Contract Functions

### Public Functions
- `create-course`: Design and publish a skill development course
- `enroll-course`: Join a course and begin the learning journey
- `certify-course`: Provide expert certification for course quality

### Read-Only Functions
- `get-course`: Retrieve course details and certification status
- `get-journey-entry`: Get specific learning journey record
- `get-enrollment-count`: Get total enrollments for a course

## Getting Started

1. Deploy the contract to Stacks blockchain
2. Register as a mentor or learner
3. Start creating courses or enrolling in skill development programs
4. Build your expertise in the decentralized learning community
